# Agent task: implement DEADGL v1.2 geometry pipe

Implement this on branch `v1.2-geometry-pipe`.

## Scope

Keep DEADGL as a C99 CPU framebuffer command machine. Do not add dependencies.

## Parser additions

Add these commands to `src/deadgl_cli.c`:

```text
camera EX EY EZ TX TY TZ FOV
face X0 Y0 Z0 X1 Y1 Z1 X2 Y2 Z2 COLOR
wireface X0 Y0 Z0 X1 Y1 Z1 X2 Y2 Z2 COLOR
```

`camera` stores scene camera state. `face` projects 3D points into screen-space `DGL_Vertex` values and calls `dgl_tri`. `wireface` projects and calls `dgl_wire_tri`.

## Math

Use local static structs/functions in `deadgl_cli.c` first. Do not expand public API unless needed.

Required math:

- vec3 subtract
- dot
- cross
- normalize
- look-at basis from eye/target/up
- perspective projection using FOV and surface height
- near plane reject for points behind/too close
