# Probe

Run:

```sh
scripts/debug_probe.sh
```

The probe builds, prints the version, runs tests, renders `examples/command_machine.dgl`, writes proof text, hashes the scene, and records file sizes.

Report:

```text
build/debug_report.txt
```

Files:

```text
build/debug_command_machine.ppm
build/debug_command_machine.proof
```
